# ⚔️ PROSAREHERE — MLBB Companion

> A dark, cinematic web dashboard built to use **while playing Mobile Legends: Bang Bang**.  
> Pick heroes, spin for random picks, build lineups, and troll your teammates — all without leaving your browser.

---

## 🎮 What Is This?

**PROSAREHERE** is a personal MLBB companion website designed to be open on the side while you're in a match or in the lobby. It's not an official app — just a slick, fast tool to help you:

- Decide **what hero to play**
- Build a **team lineup** by lane
- **Spin** for a random hero when you're feeling chaotic
- Generate a **troll pick** to mess with your squad
- Browse **all heroes** with filters

No downloads. No accounts. Just open it and play.

---

## 🧩 Features

| Feature | Description |
|---|---|
| 🎰 **Lucky Spin** | Randomly picks a hero for you — great when you can't decide |
| 🛡️ **Lineup Builder** | Assigns heroes to all 5 lanes (Gold, EXP, Mid, Roam, Jungle) |
| 🤡 **Troll Pick** | Generates cursed lane combos to chaos-draft with friends |
| 🗂️ **Hero Browser** | Filterable grid of all heroes with roles and lane info |
| 👥 **Active Players Slider** | Scrolling showcase of hero cards |
| 🎨 **Hero Banner** | Rotating cinematic hero spotlight on the homepage |

---

## 🖥️ Tech Stack

- **HTML5 / CSS3 / Vanilla JS** — no frameworks, no build tools
- **Google Fonts** — Rajdhani, Orbitron, Inter
- **Custom Fonts** — borgsquad3d, borgsquadcond, AsusROG, and more
- **CSS Variables** — full dark theme with amber-gold accent system
- **CSS Animations** — shimmer divider, flicker hero title, auto-scroll player slider

## 🚀 How to Use

1. **Clone or download** this repo
2. Open `index.html` in your browser — no server needed
3. Make sure the `Fonts/` and `image/` folders are in the same directory
4. Snap the browser window to the side of your screen while in an MLBB lobby
5. Profit

> **Tip:** Works best on a wide monitor so you can have MLBB and the site side by side.

---

## 🎨 Design

The UI follows a **dark gaming aesthetic** — think cinematic, amber-gold accents on deep navy-black backgrounds. Inspired by the visual language of high-end esports dashboards.

- **Primary color:** `#DE8F33` (amber gold)
- **Background:** `#06080b` (near black)
- **Font:** Orbitron (headers) + Rajdhani (body)
- **Vibe:** *You're about to drop into the jungle and everyone knows it.*
